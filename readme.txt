Thème personnalisé pour BTlabs - Cabinet d'étude environnementale et sociale.

== Description ==

BTlabs est un thème WordPress moderne et professionnel conçu spécifiquement pour les cabinets d'étude environnementale et sociale. Il offre une structure complète avec des types de contenu personnalisés pour les projets, services et équipe.

= Fonctionnalités principales =

* Design responsive et moderne
* Types de contenu personnalisés (Projets, Services, Équipe)
* Taxonomies personnalisées
* Zones de widgets configurables
* Support des images à la une
* Navigation personnalisée
* Formulaire de recherche intégré
* Optimisé pour les performances
* Compatible avec les dernières versions de WordPress

= Types de contenu inclus =

1. **Projets** - Pour présenter les réalisations et études
2. **Services** - Pour détailler l'offre de services
3. **Équipe** - Pour présenter les membres de l'équipe

= Zones de widgets =

* Sidebar principale
* Footer (3 zones)

== Installation ==

1. Téléchargez le thème
2. Uploadez le dossier dans `/wp-content/themes/`
3. Activez le thème dans l'administration WordPress
4. Configurez les menus et widgets

== Configuration ==

### Menus

Le thème supporte deux emplacements de menu :
- Menu Principal (header)
- Menu Footer

### Widgets

Configurez les widgets dans :
- Sidebar principale
- Footer (3 zones)

### Types de contenu

Après activation, vous trouverez dans l'administration :
- Projets
- Services  
- Équipe

### Images

Le thème définit plusieurs tailles d'images :
- btlabs-hero (1200x600)
- btlabs-card (400x300)
- btlabs-thumbnail (300x200)

== Personnalisation ==

### Couleurs

Les couleurs principales du thème sont :
- Bleu principal : #2c5aa0
- Bleu foncé : #1e3f6b
- Texte : #333
- Fond : #fff

### CSS personnalisé

Ajoutez vos styles personnalisés dans :
`assets/css/custom.css`

### JavaScript

Le fichier JavaScript principal se trouve dans :
`assets/js/main.js`
